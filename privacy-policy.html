Privacy Policy

Terakhir diperbarui: 10 Mei 2026

Auto Entry Assistant memproses data pengguna untuk kebutuhan akun, login, akses menu, validasi masa aktif, pembayaran, dan pengoperasian fitur automation.

Data Yang Diproses

Kami memproses data berikut untuk menjalankan layanan:

  • Email pengguna untuk akun, login, OTP, pemulihan password, dan komunikasi layanan.
  • Status akun, masa aktif, akses menu, versi aplikasi, dan informasi session device.
  • Data order pembayaran seperti paket, nominal, status pembayaran, dan waktu transaksi.
  • Preferensi lokal extension seperti mode run, rentang play, dan pengaturan menu.

Hal Yang Tidak Kami Lakukan

  • Kami tidak menjual data pengguna.
  • Kami tidak meminta atau menyimpan password sistem target milik pengguna.
  • Kami tidak menjalankan arbitrary remote JavaScript dari backend.

Konfigurasi workflow dari backend digunakan sebagai data selector dan pengaturan, bukan kode jarak jauh.

Data Automation

Input automation dan result diproses lokal di browser pengguna:

  • NIB/NIBEL
  • Nomor hak
  • Persil ID
  • Data SU/Hak
  • Log sukses atau gagal

Saat ini data automation tidak disimpan ke server kami. Jika di kemudian hari tersedia fitur penyimpanan cloud, pengguna akan diberitahu dengan jelas.

Layanan Pendukung

Supabase

Untuk autentikasi, database akun, akses menu, session, dan konfigurasi workflow.

Midtrans

Untuk pemrosesan pembayaran perpanjangan akun.

Brevo / SMTP

Untuk pengiriman OTP dan email transaksional.

Chrome Storage

Untuk menyimpan pengaturan lokal di browser pengguna.

Pembagian Data

Data dapat diproses oleh layanan pendukung sejauh diperlukan untuk login, pengiriman email, validasi akun, dan pembayaran.

Kami tidak membagikan data kepada pihak lain untuk penjualan data atau iklan.

Retensi Data

Data akun dan pembayaran disimpan selama diperlukan untuk operasional layanan, dukungan pengguna, audit pembayaran, dan kepatuhan hukum yang berlaku.

Data lokal di Chrome Storage dapat dihapus oleh pengguna dari browser.

Hak Pengguna & Privasi

Pengguna dapat meminta bantuan terkait data akun, koreksi data, atau penghapusan akun dengan menghubungi kontak resmi privasi kami.
