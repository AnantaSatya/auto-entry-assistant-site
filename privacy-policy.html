Privacy Policy

Kebijakan ini menjelaskan bagaimana Auto Entry Assistant memproses data saat pengguna membuat akun, login, menggunakan fitur automation, dan melakukan pembayaran perpanjangan akun.

Data yang Diproses

Kami memproses data berikut untuk menjalankan layanan:

  • Email pengguna untuk akun, login, OTP, pemulihan password, dan komunikasi layanan.
  • Status akun, masa aktif, akses menu, versi aplikasi, dan informasi session device.
  • Data order pembayaran seperti paket, nominal, status pembayaran, dan waktu transaksi.
  • Preferensi lokal extension seperti mode run, rentang play, dan pengaturan menu.

Data Automation

Input automation dan result, seperti NIB/NIBEL, nomor hak, Persil ID, data SU/Hak, serta log sukses atau gagal, diproses lokal di browser pengguna. Data tersebut tidak dikirim ke database kami kecuali jika fitur tertentu nantinya secara jelas meminta pengguna untuk menyimpan data tersebut.

Layanan Pendukung

Auto Entry Assistant menggunakan layanan pendukung berikut:

  • Supabase untuk autentikasi, database akun, akses menu, session, dan konfigurasi workflow.
  • Midtrans untuk pemrosesan pembayaran perpanjangan akun.
  • Brevo atau layanan SMTP untuk pengiriman OTP dan email transaksional.
  • Chrome Storage untuk menyimpan pengaturan lokal di browser pengguna.

Hal yang Tidak Kami Lakukan

  • Kami tidak menjual data pengguna.
  • Kami tidak meminta atau menyimpan password sistem target milik pengguna.
  • Kami tidak menjalankan arbitrary remote JavaScript dari backend.
  • Konfigurasi workflow dari backend digunakan sebagai data selector dan pengaturan, bukan kode jarak jauh.

Pembagian Data

Data dapat diproses oleh layanan pendukung di atas sejauh diperlukan untuk login, pengiriman email, validasi akun, dan pembayaran. Kami tidak membagikan data kepada pihak lain untuk penjualan data atau iklan.

Retensi Data

Data akun dan pembayaran disimpan selama diperlukan untuk operasional layanan, dukungan pengguna, audit pembayaran, dan kepatuhan hukum yang berlaku. Data lokal di Chrome Storage dapat dihapus oleh pengguna dengan menghapus data extension dari browser.

Hak Pengguna

Pengguna dapat meminta bantuan terkait data akun, koreksi data, atau penghapusan akun dengan menghubungi kontak di bawah. Permintaan akan diproses sesuai kemampuan teknis dan ketentuan hukum yang berlaku.
